# Daily Digest

_This file is automatically populated by the job-search-agent on each daily run. Do not edit manually — your changes will be overwritten._

---

The agent replaces the contents of this file each morning with:

- **New Roles Discovered** — roles found via ATS scanner or intake processing, with fit ratings and links
- **Action Required** — outreach drafts ready for your review, decisions needed
- **Follow-ups Due** — applied roles approaching the 7-day follow-up window
- **Pipeline Summary** — count by stage
- **New Learnings** — patterns from any rejection retros or interview analyses run since last digest
- **Delegation Recommendations** — suggestions for dispatching sub-agents (research, market analysis, etc.)
